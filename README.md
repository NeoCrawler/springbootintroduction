# springbootintroduction

## Description
A simple training project that uses spring boot as a back-end to hide away the open anwers from the Trivia API https://opentdb.com/api_config.php. 
The front-end is a simple WPF application that communicates with the back-end.

## Setup
### Back-end
Open up your favourite terminal in the relative directory "springbootintroduction\backend".
Run the command ".\gradlew.bat bootRun" to run the local server.

### Front-end
Open the ".sln" file in the relative directory "springbootintroduction\frontend".
Run the app through visual studio, or build the project and open the ".exe" file in the bin.
